Plataforma interna de agendamento do GER para ajudar a encontrar os melhores horários para reuniões em grupo. Facilita a consulta de disponibilidade integrada com calendários pessoais.

Built with [Vue 2](https://github.com/vuejs/vue), [MongoDB](https://github.com/mongodb/mongo), [Go](https://github.com/golang/go), and [TailwindCSS](https://github.com/tailwindlabs/tailwindcss)

## Sobre este Fork

Este é um fork do projeto [Timeful](https://github.com/schej-it/schej-it) adaptado para uso interno da equipe GER. As seguintes funcionalidades de negócio foram removidas por estarem fora do escopo da equipe:

- Sistema de doação e pagamento
- Blog e conteúdo editorial
- Integração com redes sociais públicas
- Comunidade pública (Discord, Reddit, etc.)

Todas as funcionalidades principais de agendamento e integração com calendários foram mantidas, e o projeto permanece **open-source** sob a licença AGPL v3.

## Branding e Interface

A interface do usuário está em processo de customização para representar o branding e identidade visual da equipe GER. Este trabalho está em andamento.

## Status do Desenvolvimento

Este projeto está **em desenvolvimento ativo**. Mudanças e ajustes podem ocorrer com frequência conforme as necessidades da equipe evoluem.

## Features

- Veja quando a disponibilidade de todos se sobrepoem
- Especifique facilmente intervalos de data + hora para se reunir
- Integração com Google Calendar, Outlook e Apple Calendar
- Horários "Disponível" vs "Se necessário"
- Determine quando um subconjunto de pessoas está disponível
- Agende em diferentes fusos horários
- Notificações por email + lembretes
- Duplicação de enquetes
- Grupos de disponibilidade - mantenha-se atualizado com a disponibilidade de calendário em tempo real das pessoas
- Exporte disponibilidade como CSV
- Mostre respostas apenas ao criador do evento
